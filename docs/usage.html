SHTATS

Home Statistics Usage

USAGE

shtats run [OPTIONS]

OPTIONS

-h, --help
Print the help information
-o, --output
Output filename to write the report to [default: report.html]
--since
Gather stats on all commits since this date. (Same date formatting at git see here)
--until
Gather stats on all commits until this date. (Same date formatting at git see here)
